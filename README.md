
# Project Title

Simple CRUD With Nodejs, Express, Reactjs, Mysql


## Run Locally

Clone the project

```bash
  git clone https://github.com/Abyansyah/Simple-CRUD-Nodejs.git
```

Go to the project directory

```bash
  cd backend or frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start in frontend
  nodemon in backend
```


## Build With

- NodeJs
- Express
- ReactJs
- Mysql
